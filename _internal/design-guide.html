03 Buttons

Nav CTA — ヘッダー右端ボタン
BUTTON RULE
  • Transition は必ず all .35s cubic-bezier(.4,0,.2,1)
  • Hover lift は transform: translateY(-2px)（カードは -4px〜-6px）
  • Ghost の border は 1.5px solid var(--line)、hover で var(--pink) に変色
  • 1 画面に Primary は 1 つを基本。Secondary が 2 つ以上並ぶ場合は Ghost を使う
  • Dark CTA（.btn-cta）はページ末尾の Contact 帯のみ

-2px

-4px
Service card hover
-6px
Base card hover
MOTION RULE
  • transition: all .2s 禁止。必ず .35s cubic-bezier(.4,0,.2,1) を使う
  • hover lift は transform: translateY() のみ。top/margin の変化は使わない
  • @media (prefers-reduced-motion: reduce) ですべてのアニメを停止すること
  • FV の登場アニメ: fv-rise / opacity+translateY / 遅延で順次表示
