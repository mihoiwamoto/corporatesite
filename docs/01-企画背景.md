# コーポレートサイト リニューアル企画書（背景）

— エンジニア採用強化とリブランディングに向けて —
株式会社ランステック / 2026年6月12日

※これは中長期の企画書。直近7月対応の制約は `02-議事録・決定事項.md` を優先。

## 1. 背景と目的

当社は2023年5月の合併による誕生以降、2024年11月の西原商会グループ入り、2025年6月のアローズシステムとの合併と、短期間で大きな変化を遂げてきた。現コーポレートサイトはこれらの変化を十分に反映できておらず、会社の現在地と今後の方向性が伝わりにくい。

エンジニア採用（特に中堅・経験者層）を本格化するにあたり、候補者が必ず閲覧するコーポレートサイトを「採用の武器」として再構築する。あわせて『西原商会グループの安定基盤 × 技術の先進性』というブランドイメージを確立する。

### ゴール

- 中堅エンジニア候補者の応募意欲を高め、応募数・スカウト返信率を向上させる
- 合併・グループ入りを経た「現在のランステック」のブランドストーリーを整理し発信する
- 安定感と先進性を両立したトーン&マナーへ刷新する

## 2. 現状分析

現サイトは トップ / ランステックについて / サービス / 会社概要 / 採用情報 / ニュース / お問い合わせ で構成。WordPressで運用中。

主な課題:

- **採用コンテンツの薄さ**: 求人詳細を外部媒体（engage）に依存。社員の顔・働き方・キャリアパスが見えない
- **技術情報の不在**: 技術スタック・開発事例・エンジニア文化の発信がない
- **ブランドストーリーの未整理**: 合併・グループ入りの経緯が点在し「いまのランステックは何者か」が分からない
- **情報の不整合**: 外部媒体に旧所在地（千代田区外神田）が残る等
- **技術的改善余地**: 日本語URLスラッグ、OGP・構造化データ未整備など

## 3. 採用ターゲット

実務経験3年以上の中堅エンジニア（25〜39歳）。スカウト中心の「攻め」の採用が主流で、候補者は応募前に必ずコーポレートサイトを確認する → サイトはスカウト返信率に直結する。

## 4. リブランディング方針

- **ポジショニング**: 「安定基盤 × 先進性」のハイブリッド。西原商会グループという経営基盤を持ちながら、開発・AI活用まで手がけるテック企業。
- **カラー**: 既存のブランドカラーと西原商会のグリーンを使用
- **コピー**: 誠実で平易。誇張より具体的な数字・事実
- **タグライン案（中長期）**: 「確かな基盤の上で、挑戦するエンジニアへ。」「食を支えるグループの、技術の中核へ。」「安定と挑戦は、両立できる。」

## 5. 採用コンテンツ詳細（将来）

数字で見るランステック（インフォグラフィック） / 社員インタビュー / キャリアパス・評価制度の図解 / 働く環境・制度 / 募集職種一覧（自社サイト完結） / 採用FAQ。

## 6. 実装方針（将来・WordPress継続）

- 現行WordPressを継続、テーマ新規開発またはブロックテーマで刷新
- URLスラッグ英語化（/company/ /services/ /recruit/）＋旧URLから301リダイレクト
- OGP・構造化データ（JobPosting等）・XMLサイトマップ整備
- 採用セクションはLP的な自由度、応募CTA常設
- 表示速度・スマホ最適化（閲覧7割以上がモバイル想定）

## 7. KPI

- 採用ページ閲覧数・エントリー数（公開後3ヶ月で現状比2倍）
- スカウト返信率（刷新前後比較）
- 応募者アンケート（決め手コンテンツ）
- 検索流入（指名検索CTR、関連語順位）
