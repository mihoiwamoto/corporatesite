# 実装・デザイン方針

## 直近（7月対応）

- **ゼロベースで新規作成**。既存サイトは参照しない。
- **写真不使用**: 色・図形・タイポグラフィで構成。
- **トンマナ**: LansTech基調（モーブ／ネイビー系）。安定感×先進性。
- 既存成果物 `トップページデザイン案.html`（単一HTMLファイルのモック）をベースに、事業／求人／お問い合わせページへ展開する。

### トップページモックに含まれている要素

- 西原商会グループバッジ入りファーストビュー
- 事業3カード
- 数字セクション
- 採用CTA（長谷川さんページへの導線枠を確保。公開後に有効化）
- 会社概要テーブル

## 中長期（フルリニューアル時）

- WordPress継続。テーマ新規開発またはブロックテーマで刷新。
- URLスラッグ英語化（`/company/` `/services/` `/recruit/`）＋旧URLから301リダイレクト。
- OGP・構造化データ（求人情報の JobPosting 等）・XMLサイトマップ整備。
- 採用セクションはLP的なデザイン自由度、応募CTA常設。
- 表示速度・スマホ最適化（閲覧の7割以上がモバイル想定）。
- ニュース・インタビューは社内更新できる運用設計（ブロックエディタ前提）。

## カラー方針

- 直近モック: LansTechのモーブ基調。
- 中長期企画: ネイビー基調＋シアン/グリーン系アクセント（信頼感＋先進性）。
- ※直近とフルリニューアルで方向性に差があるため、本番反映前にどちらに寄せるか要確定。

## お問い合わせフォームの技術要件

- 1フォームに統合し「お問い合わせ種別」で分岐（開発のご相談 / 採用への応募・質問 / その他）。
- 種別ごとにSlack通知チャンネルを新設して振り分け（開発→システム部、採用→人事、その他→総務【要確認】）。
- 入力項目: 種別 / お名前 / 会社名（任意）/ メールアドレス / 内容。
