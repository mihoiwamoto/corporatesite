## 4. お問い合わせページ

### 構成・方針

- フォームは1つに統合し、最初の項目「お問い合わせ種別」で振り分ける（開発のご相談 / 採用への応募・質問 / その他）
- 種別ごとの社内通知先: 開発→システム部、採用→人事、その他→総務【要確認】（Slack通知チャンネルの新設は議事録どおり）
- 入力項目は最小限: 種別 / お名前 / 会社名（任意）/ メールアドレス / 内容
- 送信後メッセージ: 「お問い合わせありがとうございます。2営業日以内にご連絡いたします。」

### リード文

> 開発のご相談、採用へのご質問など、お気軽にお問い合わせください。内容を確認のうえ、担当者より2営業日以内にご連絡いたします。
