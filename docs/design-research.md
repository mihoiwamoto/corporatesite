# BtoB SaaS企業 デザイン調査レポート

**調査日**: 2025年6月22日  
**調査対象**: LayerX, SmartHR, freee, Sansan, Notion  
**目的**: ランステックのコーポレートサイトブラッシュアップのための参考情報収集

---

## 調査サマリー

### 共通する設計原則

1. **ミニマリズム重視**: 全社が白背景ベースで装飾を最小限に抑制
2. **明確なヒエラルキー**: 見出し・本文のサイズ比が大きく、情報の優先順位が一目瞭然
3. **戦略的な余白**: 80-120px程度の垂直方向パディングで高級感を演出
4. **カード型UI**: サービス・事例紹介は一貫してカードレイアウト
5. **控えめなアニメーション**: スクロール連動やホバー効果はあるが派手ではない

---

## 1. LayerX

### ビジュアル特徴
- **カラー**: モノクロベース、赤のアクセント
- **タイポグラフィ**: 文字間スペーシングで劇的効果（「未 来 の 希 望 を」）
- **独自性**: 和英混在の極端なレタースペーシングがブランドアイデンティティ

### 学べるポイント
✅ **ミッション文の視覚的強調**  
　→ 文字間隔を広げることで記憶に残るビジュアルに

✅ **余白の贅沢な使い方**  
　→ コンテンツを詰め込まず、各セクションに呼吸感

⚠️ 注意点: 極端なミニマリズムは企業のブランドと合わない可能性あり

---

## 2. SmartHR

### ビジュアル特徴
- **カラー**: ブルー主体、パステルアクセント
- **タイポグラフィ**: クリーンなサンセリフ、明確なウェイト差
- **余白**: 80-120px垂直スペーシング（最も参考になる）
- **CTA**: オレンジ/コーラル色で視認性確保

### 学べるポイント
✅ **エンタープライズ感 + 親しみやすさの両立**  
　→ セキュリティバッジ（信頼性）+ イラスト（親近感）

✅ **CTAの反復配置**  
　→ スクロール深度に応じて複数回設置

✅ **カードシステムの統一感**  
　→ アイコン + タイトル + 説明文の一貫フォーマット

✅ **事例・社会的証明の可視化**  
　→ クライアント数、導入企業ロゴ、No.1バッジ

**→ ランステックに最も近い参考モデル**

---

## 3. freee

### ビジュアル特徴
- **カラー**: ブルー主体（信頼感）
- **タイポグラフィ**: 日本語の可読性重視の行間設定
- **CTA**: 「無料ではじめる」を複数箇所に配置
- **サービス提示**: アイコンベースのタイル型グリッド

### 学べるポイント
✅ **プライマリ/セカンダリCTAの明確な差別化**  
　→ 塗りつぶし vs アウトライン

✅ **製品エコシステムの整理**  
　→ 各サービスに独自アイコン + ブランドロゴ

✅ **背景色の交互配置**  
　→ 白 / ライトグレーで視覚的区切り

---

## 4. Sansan

### ビジュアル特徴
- **カラー**: コーポレートブルー/ティール
- **構成**: 動画デモ + 事例カード + ロゴウォール
- **余白**: 広めのサイドマージン、中央寄せコンテンツ
- **画像**: Retina対応 @2x画像 + WebP形式（パフォーマンス重視）

### 学べるポイント
✅ **デバイスモックアップでUI可視化**  
　→ 抽象的な説明を避け、実際の画面を見せる

✅ **業界別の事例分類**  
　→ IT・金融・製造など、タグで整理

✅ **グローバル/ローカル対応**  
　→ 言語切り替え、地域特化の事例

✅ **クライアントロゴの大量掲載**  
　→ トヨタ、みずほ等の有名企業で信頼性訴求

---

## 5. Notion（海外参考）

### ビジュアル特徴
- **カラー**: ディープパープル/インディゴ
- **タイポグラフィ**: Inter等のモダンサンセリフ
- **アイコン**: 絵文字スタイルで親しみやすさ
- **構成**: 機能カード + プロダクトスクリーンショット

### 学べるポイント
✅ **システマティックな繰り返し**  
　→ 矢印(→)、カードレイアウトで一貫性

✅ **98% Forbes Cloud 100等の数値的社会証明**  
　→ 具体的な実績数値の提示

✅ **プレミアムポジショニング**  
　→ 余白とタイポグラフィで高級感

## ランステックへの適用推奨事項

### 🎯 最優先で取り入れたい要素

#### 1. **余白の大幅な拡大**（SmartHR参考）
- 現在: セクションpadding 180px → **維持または120-140pxに調整**
- カード内padding: 現在56px → **適切（SmartHR並み）**
- カード間gap: 現在48px → **60-80pxに拡大検討**

#### 2. **CTAの視覚的強調**（freee/SmartHR参考）
- プライマリCTA: 現在のピンク → **そのまま維持**
- 配置: スクロール深度に応じた反復配置を追加検討
- サイズ: やや大きめのパディング（18-20px vertical）

#### 3. **サービスセクションの改善**（SmartHR/Sansan参考）
- ✅ 既に2カラムレイアウト実装済み
- 追加: 各サービスに**具体的なスクリーンショット**または**イラスト**
- 追加: 「導入事例」「実績」の可視化

#### 4. **社会的証明の強化**（Sansan参考）
- 西原商会グループのロゴ → **既にFVバッジで実装済み**
- 追加検討: グループ企業のロゴ一覧
- 追加検討: 導入システム数、稼働年数等の数値

#### 5. **タイポグラフィの最適化**（全社共通）
- 見出し: 現在68px（FV）、36px（セクション） → **適切**
- 行間: 1.5（見出し）、2.0-2.1（本文） → **SmartHR並みで良好**
- ウェイト: 300/400/500/700 → **適切な階層**

---

## 🚫 避けるべき要素

1. **過度なミニマリズム**（LayerX）  
   → システム開発会社として実績・具体性も必要

2. **多色使用**  
   → 全社がブランドカラー1-2色に絞っている

3. **派手なアニメーション**  
   → BtoB SaaSは控えめが標準

4. **長文の詰め込み**  
   → 箇条書き、カード、視覚要素で分割
